ASCII Aquarium

Source Code Sandra Shtabnaya 2019 # 106

As part of the “ASCII Ocean” series, “ASCII Aquarium” is a flipbook animation generated by converting individual frames of a coral reef video into ASCII art through the Python Pillow and ReportLab libraries. Other novels in the series include animations of ocean waves and an island coastline.

A Physical Book

Source Code Liza Daly 2017 # 105

“A Physical Book” uses the game engine Phaser to treat individual words from a scanned text as sprites, apply transformations to each letter that simulate collision, mass or acceleration, and take snapshots of the resulting animation. It produces pages of scrambled letters, which approximate chaotic movement.

Shadows

Source Code Natalia N. 2017 # 26

“Shadows” is a zine of concrete poetry created with Haskell, where each page repeats, overlays and arranges the same phrase into intricate patterns, thus arranging fifty thousand words into word art. Slow rendering in the PDF viewer animates each concrete poem, as text gradually layers across the page.
