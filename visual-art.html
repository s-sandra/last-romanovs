NNGM Entries

Year One

Source Code Hizal Celik 2016 # 148M

“Year One” is a book of 366 generated cityscapes rendered with a watercolor/painted effect. Each building represents an hour in the day. Like a bar graph, the cities depict the daily volume of text messages that the author and his girlfriend sent each other during their first year of dating.

B-9 Indifference

Source Code Eoin Noble 2017 # 100

“B-9 Indifference” uses Star Trek: The Next Generation scripts fed into a Markov chain to produce mission logs formatted to look like Star Trek LCARS files. It is meant to simulate reconstructed memories from Data, an AI from the television show.

The Several Houses of Brian, Spencer, Liam, Victoria, Brayden, Vincent, and Alex

Source Code Zach Whalen 2017 # 61

This is the HEDGEHOG that rolled up in the BALL that hit the WALL that stopped the CAR that killed the PERSON that headed for the HOUSE that Gabriel built.

This collection of children's stories, following a structure similar to the nursery rhyme "The House that Jack Built," uses ConceptNet to string together long sentences describing the relationships between multiple nouns. Each story is supplemented with images modified using ImageMagick.
