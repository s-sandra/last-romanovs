Source Code Dan Hett 2018 # 86

"Doctor Doctor" is a humorous dialogue of a hypochondriac asking medical questions to their therapist. Written in JavaScript, conversations are generated using templates and a scraped Yahoo! Webscope L6 answers dataset.
